# mario-cs50
A command-line porgram wirtten in C as part of Harvards CS50x. 

## What it does 
promts the user for a height and 
prints a right-aligned 
pyramid of that height using # symbols. 

## How to run 
1. Compile: 'make mario'
2. Run: './mario'
3. Enter a height when prompted

## Example output 
Height: 4

  ## 
 ### 
####
